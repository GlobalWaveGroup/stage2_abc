# 统计发现汇总

## Session 2026-02-28: 对称性基础统计

### 1. 对称性是真实的 (z=97.8)
- 207,729个ABC三段组合 (M1/M5/M15/H1, 各归并级别)
- L0相邻三段模长对称≥0.90: 市场~20% vs 随机~10% → **2x**
- mod_C/mod_A 在[0.8,1.2]: 真实15.8% vs 随机9.6%, **z=97.8**
- mod_C/mod_A 在[0.9,1.1]: 真实7.7% vs 随机4.9%, **z=60.3**

### 2. 分形自相似性确认
- M1/M5/M15/H1 对称比率几乎一致 (mod[.8-1.2]: 15.7%~16.0%)
- 三角形收敛/扩张 比例跨TF稳定 (~28-29%)

### 3. ABCD方向偏差 (D-segment)
- 基线 C_is_turn (D反转C): 46.3%
- mod≥0.90: **56.1%, z=12.3** — 高度显著
- 扩张+对称: C_is_turn = 58.8%, z=10.1 — 最强信号
- 收敛+对称: ~49% — 收敛趋向延续

### 4. mod_C/mod_A 不完全普适 (依赖B)
- B是趋势: mod_C/mod_A mean=3.33, median=1.82
- B是调整: mod_C/mod_A mean=0.80, median=0.57
- B是中性: mod_C/mod_A mean=1.48, median=0.93
- KS检验: 三组分布**显著不同** (p≈0)
- **但**: 用户指出这是因为约束条件没设对, 不是规律不存在

### 5. 用户核心洞察 (2026-02-28)
- **模长对称优先于幅度对称**: 应看mod_C ≈ mod_A, 不是amp_C ≈ amp_A
- **B不管是趋势还是调整都可以**: 关键是正确的AB约束条件
- **时间戳无意义**: 要的是相对位置, 不是绝对时间
- 需要用可视化案例指导建立正确的AB约束公式

### 6. 已证明的死路
- 51维聚类特征 → 无信号 (MI_real ≤ MI_shuffle)
- 60M原始15D向量 k-NN → 50% = 纯随机
- 对称计数(不看ratio向量) → 几何必然性, 不是edge
- 路径一几何预测 → 52.7%方向准确率 ≈ 随机
- 静态TP/SL策略 → z-scores -40~-216 (系统性亏损)
- 反向策略 → 也亏 (不是简单的反向)
